HOW TO USE:
━━━━━━━━━━━━━

DAGPROGRAMMA (Beheer/Admin):
1. Login as admin
2. Click "📅 Agenda Beheren" 
3. Add activities with "Toevoegen"
4. Click ✏️ to edit existing activities
5. Click 🗑️ to delete activities

NIEUWS (Admin):
1. Login as admin  
2. Click "📜 Kronieken Beheren"
3. Add: Fill form and click "Afkondigen"
4. Edit: Click "Bewerken" link on any news item
5. Delete: Click "Verwijderen" with confirmation

NEWSLETTER (Volunteers):
1. Login as admin
2. Click "📧 Nieuwsbrief Abonnees"
3. View all subscribers
4. Copy email list for newsletters
5. Delete subscribers if needed

PUBLIC SUBSCRIBE:
1. Visit /php/projectweek2/subscribe.php
2. Enter name and email
3. Click "Inschrijven op Nieuwsbrief"
4. User receives confirmation message
